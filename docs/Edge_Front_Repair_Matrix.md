# Edge 前部维修方案矩阵

> **文档定位**：MAVO Edge 8K/6K 前部维修标准方案矩阵  
> **维护者**：生产部  
> **最后更新**：2026-02-03  
> **适用范围**：Edge 8K / Edge 6K 前部更换维修

---

## 文档说明

### 业务背景

Edge 系列摄影机的前部结构和 ND 模块历经多次迭代，不同批次的机型对应不同的前部类型（F1/F2/F3）和碳纤维结构（C1/C2），配件不可混用。

为避免一线服务人员自由组合配件导致错误发货或兼容性问题，生产部制定了**标准维修方案矩阵**，系统在创建 RMA 工单时直接引用本矩阵，自动带出对应的配件清单和报价。

### 使用方式

- **系统集成**：service 系统在 RMA 工单中提供「前部维修方案」选择器
- **自动带出**：选择方案后自动填充：
  - 需要更换的配件 SKU 列表
  - 对应工时费和建议报价（区分在保/过保、国内/海外）
- **智能过滤**：根据机型和批次自动过滤仅适用的方案

## 4. 特殊规则与优惠政策

### 在保维修规则

- **保修期内**：所有方案免费（配件+工时+运费由Kine承担）
- **质量问题**：即使过保，如确认为生产批次问题，可申请免费维修

### VIP客户优惠

- **B2方案**：VIP客户可申请免费
- **其他方案**：按过保标准价格收费

### 卡口购买优惠

- **已购买电子E卡口客户**：前部更换享受8折优惠
- **英国/欧洲/美国/加拿大**：购买原生卡口可免费更换前部（B1/B2方案）

### 特殊审批

- 超出标准方案范围的维修，需生产负责人审批
- 批量召回事件：按特殊政策处理，不适用本矩阵

## 6. FAQ

### Q1: 如何判断我的机器是哪个批次？

**A**: 
1. 查看序列号前缀（系统会自动识别）
2. 查看出货日期（在产品档案中）
3. 咨询生产部（如序列号识别失败）

### Q2: 可以只换ND不换前部吗？

**A**: 
不建议。ND模块与前部结构是配套设计的，单独更换ND可能导致兼容性问题。如有特殊需求，需生产部评估后决定。

### Q3: 方案中的配件缺货怎么办？

**A**: 
1. 系统会实时显示配件库存状态
2. 如缺货，可选择等待补货或申请替代方案
3. 紧急情况可联系生产部加急生产

### Q4: 客户自己购买了前部，还能享受优惠吗?

**A**: 
客户自购配件不在本矩阵范围内。如需官方维修服务，建议使用标准方案以保证兼容性和质保。
