# 系统登录与个人设置

欢迎使用 Longhorn 售后服务系统。本章节将引导您完成首次登录并探索个人设置选项。

## 1. 系统登录

1.  **访问地址**：在浏览器（推荐使用 Google Chrome 或 Edge）中输入系统生产环境地址：`https://opware.kineraw.com`。
2.  **输入凭据**：
    *   **用户名**：由管理员为您分配的 ID 或姓名拼音。
    *   **密码**：初始密码通常由管理员提供。
3.  **身份持久化**：勾选"记住我"可以在关闭浏览器后保持会话，减少重复登录的频率。

> [!IMPORTANT]
> 登录成功后，系统会根据您的**所属部门 (Department)** 和 **角色 (Role)** 显示不同的导航菜单。

---

## 2. 个人仪表盘与概览

登录后的首个页面是**工作空间 (Workspace)**：
*   **右上角头像/工号**：点击可展开个人设置菜单。
*   **通知中心**：点击铃铛图标查看与您相关的工单变动、@提醒或 SLA 近线警告。

---

## 3. 个人中心设置

点击右上角头像选择 **"个人设置 (Profile Settings)"**：

### A. 基本信息
*   您可以查看自己的用户名、工号、所属部门及当前系统角色。
*   **多语言切换**：支持 **中文 (简体)** 与 **English**。切换后 UI 文本将立即同步更新。

### B. 账号安全
*   **修改密码**：建议首次登录后立即修改初始密码。系统要求新密码具有一定的复杂度以保证账号安全。

---

## 4. 退出登录

在公共电脑使用后，请务必点击个人菜单底部的 **"退出 (Logout)"** 以确保数据安全。

---

## 5. 故障排查

*   **忘记密码**：目前请联系系统管理员进行重置。
*   **权限显示不全**：如果您发现无法看到某些应有的菜单（如 CRM 或 部门工单），请确认您的部门设置是否正确，并让管理员检查您的角色权限。
